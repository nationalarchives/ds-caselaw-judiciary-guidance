Though content in the header or footer of a document is preserved in the PDF version of a judgment/decision, usually these elements will not show up in the web version.

Therefore, any information vital to the judgment/decision should not be placed solely in the header or footer.

Example of a document header

SSWP v AT [2022] UKUT 330 (AAC)

The parser requires a neutral citation number in order to register a judgment or decision properly. Putting this element in the header of footer of the document may prevent it from being processed.

Page numbers

If you would like page numbers to be included in the document, please use the Page Number function in the Header & Footer section in Word.
