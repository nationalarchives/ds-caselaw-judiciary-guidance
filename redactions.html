Redactions

If there are sections within a judgment or decision that need to be redacted, we recommend replacing the text with the following:

[REDACTED]

Please do not use boxes to cover up the text, these do not prevent the text from being accessed.

Example of an incorrectly displayed redaction

Bromcom Computers Plc v United Learning Trust & Anor [2022] EWHC 3262 (TCC)

Black boxes have been used in this judgment to represent redacted text but a reader can still highlight the text underneath (thankfully xâ€™s have been used). These boxes can also be removed using editing editing software so it is best not to use them at all.
