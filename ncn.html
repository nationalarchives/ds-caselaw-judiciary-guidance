Neutral citation number formatting

Most of the judgments/decisions submitted to Find Case Law should contain a neutral citation number (NCN). This is a unique reference number which aids in identifying a document.

NCNs should be styled as such:

[YEAR] COURT/TRIBUNAL number (Denomination*)
